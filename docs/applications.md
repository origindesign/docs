# Applications

## Required
- Docker (mac) or Docker for Windows
- Cygwin (https://www.cygwin.com)
- Composer (https://getcomposer.org/download/)
- Yarn (https://yarnpkg.com/en/docs/install)
- Gulp
- Node.js

Notes:
- Install Composer after Cygwin and map Cygwin's php
